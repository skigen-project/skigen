# DecisionTreeClassifier

CART decision tree classifier using Gini impurity with recursive binary splits.

$$\text{Gini}(t) = 1 - \sum_{k=1}^{K} p_k^2$$

where $p_k$ is the proportion of class $k$ samples at node $t$. At each split, the feature and threshold minimizing the weighted Gini impurity of the child nodes are selected:

$$\text{Gini}_{\text{split}} = \frac{n_L}{n} \,\text{Gini}(t_L) + \frac{n_R}{n} \,\text{Gini}(t_R)$$

Mirrors [`sklearn.tree.DecisionTreeClassifier`](https://scikit-learn.org/stable/modules/generated/sklearn.tree.DecisionTreeClassifier.html).

## Constructor

```cpp
Skigen::DecisionTreeClassifier<Scalar> tree(int max_depth = -1,
                                             int min_samples_split = 2);
```

| Parameter | Default | Description |
|-----------|---------|-------------|
| `max_depth` | `-1` | Maximum tree depth (-1 = unlimited) |
| `min_samples_split` | `2` | Minimum samples to split a node |

## Methods

| Method | Description |
|--------|-------------|
| `fit(X, y)` | Build the decision tree |
| `predict(X)` | Predict class labels |
| `score(X, y)` | Return accuracy score |

## Example

```cpp
#include <Skigen/Tree>

Skigen::DecisionTreeClassifier tree(/*max_depth=*/5);
tree.fit(X_train, y_train);
std::cout << "Accuracy: " << tree.score(X_test, y_test) << "\n";
```
