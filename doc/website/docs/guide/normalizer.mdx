# Normalizer

Row-wise normalization to unit norm (L1, L2, or max). Stateless transformer.

$$\text{L2:}\quad \hat{x}_i = \frac{x_i}{\|x_i\|_2} \qquad \text{L1:}\quad \hat{x}_i = \frac{x_i}{\|x_i\|_1} \qquad \text{Max:}\quad \hat{x}_i = \frac{x_i}{\max_j |x_{ij}|}$$

Mirrors [`sklearn.preprocessing.Normalizer`](https://scikit-learn.org/stable/modules/generated/sklearn.preprocessing.Normalizer.html).

## Constructor

```cpp
Skigen::Normalizer<Scalar> normalizer(Norm norm = Norm::L2);
```

| Parameter | Default | Description |
|-----------|---------|-------------|
| `norm` | `L2` | Norm to use: `L1`, `L2`, or `Max` |

## Methods

| Method | Description |
|--------|-------------|
| `fit(X)` | No-op (stateless) |
| `transform(X)` | Normalize each row |
| `transform_inplace(X)` | Normalize in-place |

## Example

```cpp
#include <Skigen/Preprocessing>

Skigen::Normalizer normalizer(Skigen::Normalizer<>::Norm::L2);
auto X_normed = normalizer.fit_transform(X);
// Each row now has unit L2 norm
```
