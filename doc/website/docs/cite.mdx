# How to Cite Skigen

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.19971277.svg)](https://doi.org/10.5281/zenodo.19971277)

If you use Skigen in your research or software, we would appreciate if you cite the following:

| |
|---|
| Dinh, C.: Skigen - High-Performance Machine Learning for Modern C++ and Eigen. Zenodo (2026). [DOI: 10.5281/zenodo.19971277](https://doi.org/10.5281/zenodo.19971277) |

## BibTeX

```bibtex
@software{dinh2026skigen,
  author    = {Dinh, Christoph},
  title     = {Skigen - High-Performance Machine Learning for Modern C++ and Eigen},
  year      = {2026},
  publisher = {Zenodo},
  doi       = {10.5281/zenodo.19971277},
  url       = {https://doi.org/10.5281/zenodo.19971277}
}
```

## CITATION.cff

The repository includes a [`CITATION.cff`](https://github.com/skigen-project/skigen/blob/main/CITATION.cff) file that GitHub and other tools can parse automatically. If you use GitHub's *Cite this repository* feature, the citation metadata will be pulled from that file.
